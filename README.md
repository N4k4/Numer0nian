## これは何？
 数字当てゲームヌメロン(Numer0n)をプレイするためのGUIです。
 思考エンジンと組み合わせることで、人間対エンジンやエンジン対エンジンの実行もできます。

 実態としては、Nakaが初めてGUIソフトを作るのでその練習として採用されています。
